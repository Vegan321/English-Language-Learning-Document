# Vocabulary Learning

## Prefix

1. 内外：

   - e/ex/ef：外/出

   - in/im：内/进

2. 前后：
   - pro：向前（动态）
   - pre：在前/预先/提前（静态）
   - re：回/重新/再次
3. 聚散：
   - co/con/com/col：聚集/一起/共同
   - di/dis/dif：分散/分开
4. 上下：
   - super/over/sur：上/超过
   - under/de/sub/sup：下/不足
5. 关系：
   - inter：相互之间
   - trans：转移/转变
6. 性质：
   - a + double：强调/无意义
   - un/in/im/dis/mis/ab/il：否定



## Suffix

1. Verb：
   - ate/ize/ify/ish
2. Adjective：
   - ent/ful/ous/al/ive/ible/able/ant
3. Noun：
   - ion/tion/ation/ness/ity/ment/ence/ance/ism
